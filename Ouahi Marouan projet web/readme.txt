README

L'application est une simple interface de reservation pour un hotel situé au Maroc .
Les pages web on été partiellement généré .
Les données concernant l'hotel sont soit fictives soit pris d'autres sites d'hotels juste pour etre un peu realiste .
L'utilisateur choisi une période a réservé et combien de jours il souhaiterai sejourner a l'hotel
Les données sont ensuite insérer dans la base de données
Les données de réservation sont a visualiser directement sur la base de donnée.
